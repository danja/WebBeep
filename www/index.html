Make Beeps

Enter any text (up to 63 characters) and click the Make Beeps button to get the download.

Web addresses (URLs) work well - now you can give yourself a URI you can hear!

Here's a sample: Example.mp3

Decode Beeps

Click Browse to locate a Web Beep mp3 file on your computer, then click Decode to see the text it contains.

This is a new alpha service, any problems, let me know.
